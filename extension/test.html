Troubleshooting

If context menu options do not appear:

  1. Open chrome://extensions
  2. Find FACTORIZER and click "Details"
  3. Make sure "Site access" is set to "On all sites"
  4. Enable "Allow in incognito"
  5. Click the refresh/reload icon
  6. Restart your browser

Open the developer console (F12) before using this page to see debug messages.
